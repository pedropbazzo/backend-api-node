## :cd: How to run local

At the root of your project create an .env file with the following contents:

```javascript
JWT_SECRET=<YOUR_SECRET>
JWT_EXPIRE=30d // or any reasonable value that you prefer
MONGOURI=<YOUR_DB_CONNECTION_URI>
```

```
- npm install  
- npm run start
```
